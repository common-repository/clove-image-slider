Simple and easy to use but powerful image slider with in depth customizations.

== Description ==

**Clove Image Slider** is a focus more on usability and simplicity of image slider. clove image slider provides the most used type of image slider layouts.

### Clove Image Slider Features

* Mulitple galleries.
* Different layouts for slider.
* layout customizations.
* Shorcode and theme code to show image slider.
* Custom fonts.
* Image preloader.


== Installation ==

**Clove Image Slider [Installation Guide]**

1. You can:
 * Upload the entire `clove-image-slider` folder to the `/wp-content/plugins/` directory via FTP.
 * Upload the zip file of plugin via *Plugins -> Add New -> Upload* in your WordPress Admin Panel.
 * Search **Clove Image Slider** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate the plugin through the *Plugins* menu in WordPress Admin Panel.
3. Click on *Clove Image Slider* to see the galleries and new ones.
4. Ready, now you can use it.

== Frequently Asked Questions ==

= How to create a slider? =
1. First go to clove image slider and click on add new image slider.
2. click on add slide.
3. You can upload images or select from media.
4. You can hold control and select Mulitple images.
5. Now publish the slider and copy Shorcode to your page.
